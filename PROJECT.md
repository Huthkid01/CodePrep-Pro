# CodePrep Pro - Technical Interview Prep Platform

## Project Name
**CodePrep Pro**

---

## Problem Statement
Software developers preparing for technical interviews often struggle with:
- Finding quality coding challenges
- Practicing under time pressure
- Tracking their progress over time
- Accessing solutions for learning

**CodePrep Pro** solves these problems by providing a comprehensive, interactive coding practice platform with built-in timers, progress tracking, and instant feedback.

---

## Target Users
- Software developers preparing for technical interviews
- Computer science students practicing algorithms
- Programmers looking to improve problem-solving skills
- Anyone wanting to practice coding under pressure

---

## Features List

### Core Features (8 Functional Features)
1. **15+ Coding Challenges** - LeetCode-style problems across 3 difficulty levels (Easy, Medium, Hard)
2. **Countdown Timer** - Per-challenge timer with pause/resume/reset functionality
3. **Difficulty Filters** - Filter challenges by Easy, Medium, or Hard
4. **Code Editor** - Built-in JavaScript editor with line numbers
5. **Hint System** - Progressive hints (costs 30 seconds each)
6. **Solution Viewer** - View complete solutions for all challenges
7. **Progress Tracking** - localStorage-based history of solved challenges
8. **Statistics Dashboard** - Track solved count, time spent, success rate, and streak

### Additional Features
- Auto-pause timer when tab loses focus
- Score calculation based on time and hints used
- Responsive design for mobile and desktop
- Visual difficulty badges
- Smooth animations and modern UI

## How to Use
1. Open the deployed site
2. Browse challenges in the sidebar (filter by difficulty)
3. Click a challenge to start the timer
4. Read the problem description
5. Write your solution in the code editor
6. Use hints if needed (costs time)
7. Submit to check your answer
8. View the solution to learn
9. Track your progress in the stats dashboard
